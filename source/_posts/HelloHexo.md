##### 多端同步编辑步骤
```
1. git clone //克隆到本地
2. npm install //安装必须的库包文件
3. hexo clean //清除编译缓存文件
4. 重新编辑
5. hexo s //本地预览
6. git add . //将修改添加至本地
7. git commit -m "" //提交修改注释
8. git push  //推送远程
```
